## 強み・差別化ポイント

### 1. 自爪育成の圧倒的な専門性
- **一般ネイルとの根本的な違い:**

| 一般ネイル | 自爪育成 |
|---|---|
| デザイン中心 | 構造中心 |
| 当日完成 | 数ヶ月設計 |
| 表面（見た目） | 爪母・爪床（根本） |
| 単発来店 | 継続型モデル |
| 「可愛い」を売る | 「人生が変わる」を売る |

- **専門知識の深さ:**
  - 構造学: ケラチン構造、水分量12〜16%、水分と油分のバランス、ターンオーバー
  - 技術: ハイポイントの位置、ドーム構造、応力分散、先端厚み設計（「盛る技術」ではなく「壊れない構造を作る技術」）
  - 生活設計: 圧・水・摩擦・薬剤・温度差の5要素による爪ダメージ管理
  - 補正力: 反り爪、横広がり、深爪、剥離、噛み癖 → 「癖を読む力」がある人だけが本当の育成ができる

### 2. 独自のターゲット定義
- 「ネイルが好きな人」ではなく「**ネイルができなかった人**」
  - 深爪・噛み癖で他サロンに断られた人
  - 爪が薄くてジェルができない人
  - 爪にコンプレックスを持つ人
- この層は"価格"より「解決できるかどうか」で選ぶ → 価格競争にならない

### 3. 実体験に基づく説得力
- 自身が3,000円赤字サロンから年商2,000万に至ったリアルストーリー
- 「教科書的な知識」ではなく「自分が通ってきた道」を教えられる
- 4年間の苦悩→突破の体験が、受講生の共感と信頼を生む

### 4. 悩み解決型ビジネスモデル
- 専門性が高く、再現性に理論が必要 → 参入障壁が高い
- 悩み解決型 → 価格ではなく「解決できるかどうか」で選ばれる
- 継続型 → 月1回来店、3〜6ヶ月設計でLTVが高い

### 5. 集客設計の仕組み化
- 設計型ハイブリッド集客（無料×有料の組み合わせ）
- ホットペッパー活用で新規流入80%以上を獲得
- 口コミ設計三原則: 感動ポイントを意図的に作る、変化を言語化する、次回来店前に依頼する
- 4チャネル導線: 有料媒体（即時獲得）、SNS（信頼醸成）、LP（サービス理解）、Googleマップ（近隣獲得）
