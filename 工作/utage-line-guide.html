📲 UTAGE × LINE連携 設定ガイド

初めてでも迷わない！3ステップで完了

💡 そもそも何をやるの？

💬 LINE公式
アカウント
🔗
⚙️ UTAGE

LINE公式アカウントの「鍵」を UTAGE に渡して、
UTAGEからLINEメッセージを自動配信できるようにします。

1
準備する
📍 LINE側で作業

LINE公式アカウントで「Messaging API」をオンにして、接続用の鍵を取得します。

💬 Messaging APIって？
LINE公式アカウントを外部ツール（UTAGE）から操作できるようにする機能。これをオンにしないとUTAGEと繋がりません。
  1. 1 🔐
    LINE公式アカウント管理画面にログイン
    連携したいアカウントを選択 → 右上の⚙設定
  2. 2 🔌
    Messaging API を有効化
    左メニュー「Messaging API」→「利用する」をクリック
  3. 3 📁
    プロバイダーを作成（または選択）
    初めての場合は新規作成。名前とメールを入力
  4. 4 🔑
    Channel ID と Channel Secret をコピー
    表示された2つの値を必ずコピー保存！
💬 Channel ID / Channel Secret って？
UTAGEが「このLINEアカウントを操作していいよ」と認証するための鍵（パスワードのようなもの）です。
🔐 LINEログインチャネルも作成

友だち追加の経路を正確に判定するために、もう1つチャネルを作ります。

  1. 5 🌐
    LINE Developers で「LINEログイン」を新規作成
    アプリタイプは「ウェブアプリ」を選択
  2. 6 🟢
    「開発中」→「公開」に変更
    忘れがち！ ボタンをクリックして公開状態にする
  3. 7 🔑
    チャネルID と チャネルシークレットをコピー
    Messaging APIの鍵とは別の値です！
⬇️
2
接続する
📍 UTAGE側で作業

さっきコピーした4つの鍵を UTAGE に貼り付けます。

  1. 1 📋
    UTAGEでLINEアカウント設定を開く
    メール・LINE配信 → LINEアカウント → LINEアカウント設定
  2. 2 📝
    4つの値を貼り付ける
🔑 Channel ID
（Messaging API）
🔑 Channel Secret
（Messaging API）
🔐 チャネルID
（LINEログイン）
🔐 チャネルシークレット
（LINEログイン）
⚠️ 4つとも全部違う値！
Messaging API用とLINEログイン用で、それぞれ別のIDとシークレットがあります。混同しないよう注意。
  1. 3 💾
    「保存」をクリック
    Webhook URLが自動で設定されます
📡 LINE側のWebhook設定
  1. 4 📡
    LINE管理画面で「応答設定」を開く
    Webhookを「オン」（緑）にする
  2. 5 ✅
    Webhook URLが入っていることを確認
    UTAGEのURLが自動入力されていればOK
💬 Webhookって？
LINEで友だち追加やメッセージ受信があった時、その情報をUTAGEに自動で通知する仕組みです。
⬇️
3
確認する
📍 UTAGE + スマホ

テスト送信して、ちゃんと繋がっているか確認します。

  1. 1 📄
    UTAGEでテスト用シナリオを作成
    シナリオ追加 → LINEメッセージを1通追加
  2. 2 ⏱️
    送信タイミングを「登録直後」に設定
    友だち追加したらすぐメッセージが届くようにする
  3. 3 📱
    QRコードをスマホで読み取り
    LINE登録ページのQRを読み取り → 許可 → 友だち追加
  4. 4 💬
    メッセージが届けば成功！
    トーク画面でテストメッセージを確認
🎉 メッセージが届いたら連携完了！
⚠️ よくあるミス・注意点
  • 4つの値（ID×2、シークレット×2）は全部異なる値です。混同注意！
  • 値は手入力せず、必ずコピーボタンで貼り付け（スペース混入防止）
  • LINEログインチャネルを「公開」にし忘れるとエラーになります
  • 1つのLINE公式アカウントに紐づけられるUTAGEアカウントは1つだけ
  • 保存時に500エラーが出たら、値の入力ミスを確認してください
