## セクション1: 表紙

```
[クライアント名] 様
YouTube戦略レポート
── チャンネル分析 × コンテンツ戦略 × SNS展開 ──

  ↓ 「詳しくはプロフィールのリンクから」

  ↓ 「LINE登録で○○プレゼント」

  ↓ 「無料相談はこちら」

```

各ステップに:

- KPI（目安の数字）
- 具体的なアクション

---

## トーンの注意点

- **提案書のトーン**: 「分析結果に基づく根拠のある提案」という専門家ポジション
- **断定しすぎない**: 「〜が効果的です」「〜をおすすめします」（「〜すべきです」は避ける）
- **数字で語る**: 感覚ではなくデータ・事例ベース
- **クライアントの良いところを最初に**: 改善提案の前に必ず強みを認める
- **実行可能な提案**: 理想論ではなく、今すぐ始められることを中心に

## 特典企画書（bonus-plan.html）の詳細構成

### 表紙
```
[クライアント名] 様
特典コンテンツ企画書
── LINE登録特典 × リード獲得 × ファン化 ──
```

### なぜ特典が必要か（1スライド）
- YouTube視聴者 → LINE登録の転換率を上げるため
- 「無料で価値提供」→「もっと知りたい」→「個別相談」の流れ
- 特典の良し悪しがLINE登録率を3〜10倍変える

### 特典案カード（各1スライド × 3案）

各カードの構成:
```
┌─────────────────────────────────┐
│ 💡 特典案① [タイトル]           │
│                                 │
│ 形式: PDF（全○ページ）          │
│                                 │
│ ┌─ 構成 ──────────────────────┐ │
│ │ 第1章: ...                   │ │
│ │ 第2章: ...                   │ │
│ │ 第3章: ...                   │ │
│ └─────────────────────────────┘ │
│                                 │
│ 🎯 刺さる理由                   │
│ YouTube分析で○○というニーズが    │
│ 確認された。この特典は...        │
│                                 │
│ 制作難易度: ★★☆               │
│ 制作期間: 約○週間               │
└─────────────────────────────────┘
```

### おすすめ案（1スライド）
- 3案の比較表
- 推薦案とその理由
- 「迷ったらまずこれ」

### 次のステップ（1スライド）
1. 特典の方向性を決定
2. 特典コンテンツの制作
3. LINE公式アカウント構築
4. YouTubeからの導線設計
5. 運用開始

## 特典タイプの引き出し

クライアントのジャンルに応じて使い分ける:

| タイプ | 例 | 向いているジャンル |
|--------|----|--------------------|
| PDFガイド | 「○○完全ガイド」 | 教育系・専門系全般 |
| チェックリスト | 「○○セルフチェック10項目」 | 美容・健康・ビジネス |
| 動画講座 | 「非公開○○セミナー」 | コーチング・コンサル |
| 診断ツール | 「あなたの○○タイプ診断」 | 美容・性格・ビジネス |
| テンプレート | 「そのまま使える○○テンプレ」 | ビジネス・マーケ |
| 事例集 | 「○○ビフォーアフター集」 | 美容・ダイエット・教育 |
| ワークシート | 「○○を見つける7つの質問」 | 自己啓発・コーチング |
