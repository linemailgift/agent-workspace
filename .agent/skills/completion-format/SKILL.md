# 完了フォーマットスキル（completion-format）

## 最上級ルール（全ルールに優先）

このルールは他の全てのルール・スキルより優先される。

## ルール

### 1. URLは必ず一番下に出す
- 回答の最後の方にURLを出す
- URLより下には完了時刻以外何も書かない

### 2. 案内・説明はURLの上に書く
- 補足、注意点、次のステップの案内、質問など → 全てURLより上
- URLの後に「他にも〜」「ちなみに〜」など絶対に書かない

### 3. 完了時刻は必ず書く
- URLがある場合: URL → 完了時刻 で終了
- URLがない場合: 回答の最後に完了時刻を書く
- 全てのタスク完了時に必ず書く

## 出力テンプレート

### URLがある場合

```
（案内・説明・補足・質問は全部ここ）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
https://linemailgift.github.io/agent-workspace/xxxxx
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

完了時刻: 2026-XX-XX HH:MM
```

### URLがない場合

```
（回答本文）

完了時刻: 2026-XX-XX HH:MM
```

## やってはいけないこと

- URLの後に説明や補足を書く
- URLの後に「他にありますか？」等の質問を書く
- 完了時刻を書き忘れる
- URLより上に別のURLを目立つ形で出す（メインURLが埋もれる）
