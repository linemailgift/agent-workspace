# 議事録作成ルール

## 基本ルール
1. **ちいくじ = ユーザー本人**。「氏」は付けない。「ちいくじ氏」→「ちいくじ」に統一
2. 「知育時」「チーフジ」等の誤変換 → 「ちいくじ」に修正
3. **クライアントには敬称「さん」を付ける**（例：おがたつさん）。呼び捨て禁止
4. **UTAGE** はシステム名称。「宴（UTAGE）」「宴」ではなく **UTAGE** とだけ表記
5. 金額は控えめに表示（大きいフォントや目立つ装飾は避ける）。クライアント向け資料なので驚かせない

## スライド作成ルール

### デザイン
- **白背景 + ゴールド（#c9a96e）アクセント** の高級感あるデザイン
- フォント: Noto Sans JP + Cormorant Garamond（番号・英語）
- カード: 背景 #fafaf8、ボーダー #eee、上部にゴールドライン
- ハイライトボックス: 背景 #faf8f3、左ボーダー #c9a96e
- テーブル: ヘッダー背景 #faf8f3、ゴールド文字
- タグ: ボーダー #ddd、ゴールドタグは border #c9a96e + 背景 #faf8f3

### アニメーション
- スライド切り替え: **上から下** に降りてくるアニメーション
  - 非アクティブ: `transform: translateY(-30px); opacity: 0;`
  - アクティブ: `transform: translateY(0); opacity: 1;`

### 構成
- ナビゲーション: 右下に←→ボタン、左下にページ番号
- 矢印キー・スペースキー・スワイプ対応
- Section番号は `Cormorant Garamond` フォントで英語表記

## 完了時のルール
1. チャットに **完成時刻** を表示（ファイル内ではなくチャット上）
2. 完了音を鳴らす: `afplay -r 0.5 /System/Library/Sounds/Submarine.aiff`
