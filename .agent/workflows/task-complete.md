# タスク完了時の必須アクション

すべてのタスクが完了したら、**必ず以下の2つを実行すること。例外なし。**

## 1. 完了音を鳴らす

// turbo
```bash
afplay -r 0.5 /System/Library/Sounds/Submarine.aiff
```

## 2. 完成時刻を表示する

チャット上に以下のフォーマットで表示する：

```
完成時刻：**HH:MM（JST）** — YYYY年MM月DD日
```

- 現在のローカル時刻をそのまま使う
- ファイル内には書かない。チャット上に表示するだけ
- 成果物（URL等）がある場合はその直後に表示する
